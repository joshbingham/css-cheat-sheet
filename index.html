CSS Selectors
Selector Explanation Example
Universal selector: * Selects all elements * { margin: 0; padding: 0; }
(resets margin and padding for all elements)
Element selector: h1 Selects all h1 elements h1 { color: green; }
(sets text color to green for all h1 elements)
Class selector: .class Selects elements with class "class" .header { background-color: #f2f2f2; }
(sets background color for elements with class "header")
ID Selector: #id Selects element with id "id" #logo { width: 100px; height: 100px; }
(sets width and height for element with id "logo")
CSS Properties
Color & Background
Property Explanation Example
color: Sets text color color: #333;
(sets text color to dark gray)
background-color: Sets background color background-color: #f2f2f2;
(sets background color to light gray)
background-image: Sets background image background-image: url('image.jpg');
(sets background image tp "image.jpg")
Typography
Property Explanation Example
font-family: Sets font family font-family: Arial, sans-serif;
(sets font family to Arial with sans-serif as a fallback)
font-size: Sets font size font-size: 16px;
(sets font size to 16 pixels)
text-align: Sets text alignment text-align: center;
(centers text)
Layout
Property Explanation Example
display: Sets display type (block, inline, flex, etc.) display: flex;
(sets display to flexbox)
position: Sets positioning (static, relative, absolute, etc.) position: absolute;
(sets positioning to absolute)
top, right, bottom, left: Sets offset properties top: 10px; left: 20px;
(sets top and left offsets)
